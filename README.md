## 🛠️ Tecnologías y Herramientas
Para este desarrollo se han utilizado herramientas líderes en la industria para garantizar velocidad y diseño:

* **WordPress:** Como motor principal de gestión de contenidos.
* **Elementor:** Constructor visual para crear un diseño personalizado y moderno.
* **Astra Theme:** Tema base ligero y altamente optimizado.
* **WPvivid:** Sistema de copias de seguridad y migración al servidor.
* **InfinityFree:** Hosting utilizado para el despliegue del sitio en vivo.

## 📁 Estructura del Proyecto
Este repositorio contiene la carpeta `wp-content`, que es el corazón de la personalización del sitio:
* `/themes`: Plantillas y estilos visuales.
* `/plugins`: Funcionalidades añadidas para la gestión de clientes.
* `/uploads`: Imágenes, logos y recursos multimedia de la firma.
